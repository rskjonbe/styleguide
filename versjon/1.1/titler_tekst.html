Titler og tekst

Skalerbarhet

Fontstørrelse skal angis i em/rem (relativ enhet). Ved nyutvikling og oppgradering av CSS anbefaler vi følgende veiledende standard for mediaskalerbare titler og tekst fra Typecast.com. CSS-koden er gjengitt i avsnittene under.

Body

body { font-size: 100%; } body, h1, h2, h3, h4, h5, h6 { font-size-adjust: 0.5; } /* Body Content */ #page { font-size: 1em; /* equivalent to 16px */ line-height: 1.25; /* equivalent to 20px */ } @media (min-width: 43.75em) { #page { font-size: 1em; /* equivalent to 16px */ line-height: 1.375; /* equivalent to 22px */ } }

Heading 1

h1 { font-size: 2em; /* 2x body copy size = 32px */ line-height: 1.25; /* 45px / 36px */ } @media (min-width: 43.75em) { h1 { font-size: 2.5em; /* 2.5x body copy size = 40px */ line-height: 1.125; } } @media (min-width: 56.25em) { h1 { font-size: 3em; /* 3x body copy size = 48px */ line-height: 1.05; /* keep to a multiple of the 20px line height and something more appropriate for display headings */ } }
